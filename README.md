
### Тест Websocket:

Тестовый проект по работе с FastAPI и WebSocket.
Для разных подключений создаются разные ID пользователей. Можно отправлять сообщения и видеть их в реальном времени для всех пользователей.

## Технологии

В проекте применены следующие технологии:

* Python
* FastAPI
* WebSocket
